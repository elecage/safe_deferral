## PN-02: Lighting Control Node

**GPIO 매핑**

| 신호 | GPIO | 비고 |
|------|------|------|
| LIVING_ROOM_LIGHT | GPIO4 | 릴레이 제어 출력 |
| BEDROOM_LIGHT | GPIO5 | 릴레이 제어 출력 |

**추가 부품**

| 부품 | 수량 | 비고 |
|------|------|------|
| 5V 릴레이 모듈 (옵토커플러 내장) | 2 | SRD-05VDC-SL-C 계열 |
| LED (테스트용 부하) | 2 | 실제 조명 없을 때 |
| 220Ω 저항 | 2 | LED 전류 제한 |

**특수 회로**: 릴레이 모듈이 3.3V 트리거를 지원하지 않을 경우 NPN 드라이버 회로 추가.

```
GPIO4/5 ──[1kΩ]──► BC547 Base
                   Collector ── Relay Coil(+) ── 5V
                   Emitter ── GND
                   Relay Coil 양단: 1N4148 플라이백 다이오드 (역방향)
```

> 대부분의 옵토커플러 내장 릴레이 모듈은 3.3V 신호로 동작하므로 구매 전 사양 확인.

## PN-03: Environmental Context Node

**GPIO 매핑**

| 신호 | GPIO | 비고 |
|------|------|------|
| DHT22 DATA | GPIO1 | 1-Wire |
| I2C SDA (BH1750) | GPIO2 | |
| I2C SCL (BH1750) | GPIO3 | |
| PIR OUTPUT | GPIO4 | |

**추가 부품**

| 부품 | 수량 | 비고 |
|------|------|------|
| DHT22 (온도/습도 센서) | 1 | 1-Wire 프로토콜 |
| 10kΩ 저항 | 1 | DHT22 DATA 풀업 — **필수** |
| BH1750 모듈 (조도, I2C) | 1 | I2C 주소 0x23 또는 0x5C |
| HC-SR501 PIR 모듈 (재실감지) | 1 | 3.3V 또는 5V 동작 |
| 10μF 전해 커패시터 | 1 | 전원 안정화 |

**특수 회로**: DHT22 DATA 라인 풀업 저항 필수.

```
VCC ──[10kΩ]──┬── DHT22 DATA ── GPIO1
              └── (풀업)

풀업 저항 생략 시 읽기 오류 다발 발생.
```


## PN-05: Gas / Smoke / Fire Evidence Node

**GPIO 매핑**

| 신호 | GPIO | 비고 |
|------|------|------|
| MQ-2 ANALOG OUT | GPIO0 (ADC) | 전압 분배기 필수 |
| MQ-2 DIGITAL OUT | GPIO1 | 디지털 임계 출력 (선택) |

**추가 부품**

| 부품 | 수량 | 비고 |
|------|------|------|
| MQ-2 모듈 (가스/연기) | 1 | 아날로그(A0) + 디지털(D0) 출력 |
| 10kΩ 저항 | 1 | 전압 분배기 상단 |
| 20kΩ 저항 | 1 | 전압 분배기 하단 |
| 100μF 전해 커패시터 | 1 | MQ-2 히터 전류 보상 |

**특수 회로**: MQ-2 아날로그 출력(0~5V)을 ESP32 ADC(최대 3.3V)에 연결하기 위한 전압 분배기 필수.

```
MQ-2 A0 (0~5V) ──[10kΩ]──┬── ESP32 ADC GPIO
                           [20kΩ]
                           GND

출력 전압 = 5V × 20 / (10+20) = 3.33V ≤ 3.3V ✓
```

> MQ-2 디지털 출력(D0)만 사용하는 경우 전압 분배기 불필요. 단, 임계값은 모듈 가변저항으로만 조정.

**전원 주의**: MQ-2 히터 예열 약 30초 필요. 히터 전류 ~150mA. USB 단독 공급 시 불안정 가능 — 500mA 이상 5V 공급 권장.

| 신호 | GPIO | 비고 |
|------|------|------|

| I2C SCL (MPU-6050) | GPIO3 | |

**추가 부품**

|------|------|------|

**낙상 감지 알고리즘 파라미터 (코드 기준)**

| 파라미터 | 값 |
|----------|-----|

**장착 위치 중요**: 낙상 감지 정확도는 센서 부착 위치에 크게 의존. 허리 또는 가슴 부위 부착 권장.

|------|------|------|

| STATUS_LED | GPIO7 | |

**추가 부품**

| 부품 | 수량 | 비고 |
|------|------|------|
| 패시브 부저 (수동형) | 1 | **능동(액티브) 부저 사용 불가** — LEDC PWM 필요 |
| BC547 NPN 트랜지스터 | 1 | 부저 GPIO 드라이버 |
| 1kΩ 저항 | 1 | 베이스 전류 제한 |
| 1N4148 다이오드 | 1 | 역기전력 보호 |
| LED (상태 표시) | 1 | |
| 220Ω 저항 | 1 | LED 전류 제한 |
| (선택) DFPlayer Mini 모듈 | 1 | MP3 TTS 출력 |
| (선택) 소형 스피커 8Ω/0.5W | 1 | DFPlayer Mini용 |
| (선택) microSD 카드 | 1 | 음성 파일 저장 |

**특수 회로**: 부저 NPN 드라이버 필수 (GPIO 직결 시 최대 40mA 초과 → ESP32 손상 위험).

```
GPIO6 ──[1kΩ]──► BC547 Base
                 Collector ──[부저(+)]
                 Emitter ── GND
                 부저(-) ── 3.3V 또는 5V
                 부저 양단: 1N4148 (역방향)
```

**GPIO 매핑**

| 신호 | GPIO | 비고 |
|------|------|------|
| LOCK_RELAY | GPIO4 | HIGH = 잠금 상태 |
| STATUS_LED | GPIO5 | |

**추가 부품**

| 부품 | 수량 | 비고 |
|------|------|------|
| 5V 릴레이 모듈 (옵토커플러 내장) | 1 | |
| LED (상태 표시) | 1 | |
| 220Ω 저항 | 1 | LED 전류 제한 |
| (실험 데모용) SG90 서보모터 | 1 | 실제 잠금장치 대신 동작 시뮬레이션 |
| (실제 잠금장치) 솔레노이드락 | 1 | 12V DC |
| (실제 잠금장치) 12V DC 어댑터 | 1 | 솔레노이드 전원 |
| (실제 잠금장치) IRF540N MOSFET | 1 | 솔레노이드 구동 |
| (실제 잠금장치) 1N4007 다이오드 | 1 | 솔레노이드 역기전력 보호 |

**특수 회로**: 실제 솔레노이드락 사용 시 MOSFET 드라이버 및 별도 12V 전원 필요.

```
GPIO4 ──[1kΩ]──► IRF540N Gate
                 Drain ── 솔레노이드(+) ── 12V
                 Source ── GND
                 솔레노이드 양단: 1N4007 (역방향)
```
