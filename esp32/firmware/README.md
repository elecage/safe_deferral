# esp32/firmware

Future location for ESP32 firmware projects, templates, build-system files, and board-specific defaults.

No production firmware project is maintained here yet. Current build-readiness validation uses the ESP-IDF `hello_world` sample copied into the local `~/esp32_workspace` by `esp32/scripts/configure/30_prepare_sample_project_esp32.*`.

When firmware templates are added, they should remain bounded physical node templates and should not embed safety-critical policy decisions.
